# AI-
本仓库是一款AI 文本总结工具，专注于利用人工智能技术实现文本内容的智能提炼与概括，支持对文章、段落、对话等各类文本快速生成精简、准确的摘要，帮助用户高效获取核心信息，提升阅读与信息处理效率。
# AI文本总结工具

一句话说明：

输入一段文本，自动生成简单总结。

---

## 项目介绍

当文章内容较长时，可以通过该工具快速生成简单总结。

适用于：

- 学习笔记
- 文章阅读
- 内容整理

---

## 安装方法

### 下载项目

1. 打开项目仓库页面  
[https://github.com/wujy309-jpg/AI](https://github.com/wujy309-jpg/AI-.git)

2. 点击绿色 **Code** 按钮

3. 点击 **Download ZIP**

4. 解压文件到电脑

--

## 使用方法

1.进入下载好的文件夹后使用shift+右键运行终端

2.输入：python summarizer.py 即可启动
